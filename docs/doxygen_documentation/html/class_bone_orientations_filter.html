Detailed Description

Implementation of a Holt Double Exponential Smoothing filter for orientation. The double exponential smooths the curve and predicts. There is also noise jitter removal. And maximum prediction bounds. The parameters are commented in the Init function.

Initialize the filter with a set of manually specified TransformSmoothParameters.

Parameters
smoothingValueSmoothing = [0..1], lower values is closer to the raw data and more noisy.
correctionValueCorrection = [0..1], higher values correct faster and feel more responsive.
predictionValuePrediction = [0..n], how many frames into the future we want to predict.
jitterRadiusValueJitterRadius = The deviation angle in radians that defines jitter.
maxDeviationRadiusValueMaxDeviation = The maximum angle in radians that filtered positions are allowed to deviate from raw data.
